Télécharger l'application Android

MedLink n'est pas distribuée sur le Google Play Store. Le fichier d'installation (APK) est proposé directement en téléchargement ci-dessous.

Télécharger l'APK MedLink

Cette installation se fait hors du Play Store. Android affichera un avertissement à ce sujet lors de l'installation : c'est normal, il suffit de suivre les étapes ci-dessous.

Installation sur votre téléphone Android

  1. Téléchargez le fichier avec le bouton ci-dessus, depuis le navigateur de votre téléphone.
  2. Ouvrez le fichier téléchargé depuis vos notifications ou l'application « Fichiers ».
  3. Si Android bloque l'installation, ouvrez Paramètres, puis autorisez « Installer des applications inconnues » (ou « Sources inconnues » sur les versions plus anciennes d'Android) pour l'application utilisée pour ouvrir le fichier (navigateur ou gestionnaire de fichiers).
  4. Relancez l'installation, puis confirmez.
